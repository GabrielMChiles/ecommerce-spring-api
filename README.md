# ecommerce-spring-api
API REST desenvolvida em Java e Spring Boot para simular o backend de um E-commerce (Sistema de Gestão de Pedidos). 
O projeto aplica a arquitetura em 3 camadas (Controller, Service, Repository), Programação Orientada a Objetos (POO) e modelagem de dados relacional utilizando JPA/Hibernate. Principais Funcionalidades e Regras de Negócio:
* Gerenciamento completo (CRUD) de Clientes, Produtos e Pedidos.
* Controle de estoque rigoroso durante a criação de pedidos.
* Cálculo automatizado do valor total dos pedidos.
* Máquina de estados para o fluxo do pedido (CRIADO, PAGO, ENVIADO, CANCELADO).
Projeto desenvolvido como requisito de avaliação da N1  da disciplina de Construção de Software.
